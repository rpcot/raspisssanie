# raspisssanie

Телеграм-бот и апи для веб-сайта для удобного просмотра и управления расписанием и домашними заданиями в школе.

## О проекте

❤️‍🔥 Этот бот помогает быстро получать актуальное расписание и домашние задания.

🕐 Бот сохраняет все предыдущие, отображает текущую и две следующих недели.

✏️ Расписание заполняется учениками, поэтому можно указывать все детали: домашние задания, проверочные, отмены уроков, внеурочные мероприятия и так далее.

## Что можно указывать в боте

- Добавление, удаление и перенос домашних заданий
- Добавление, удаление, отмена и перенос уроков  
- Внеурочные мероприятия и примечания к конкретным дням  
- Изменение расписания звонков  
- Изменение кабинетов, в которых проходят уроки  
- Отметка дня как выходного  
- Отображение времени начала и окончания занятий  
- Запись проверочных, контрольных, тестовых и других видов контрольных работ

## Основные команды

- /start — приветствие и краткая информация  
- /help — список команд  
- /site — ссылка на сайт  
- /today — расписание на сегодня
- /tomorrow — расписание на завтра

## Запуск проекта

0. Клонировать репозиторий:  
```bash
git clone https://github.com/rpcot/raspisssanie.git
cd raspisssanie
```

1. Установить yarn, если его нет:
```bash
npm install --global yarn
```

2. Установить зависимости:  
```bash
yarn install
```

3. Запустить бота:  
```bash
yarn start
```

4. Запустить API:  
```bash
yarn api-start
```
